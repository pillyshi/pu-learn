## Quick Start

Install as a project:

```
git clone https://github.com/pillyshi/pu-learn.git
cd pu-learn/
pipenv install
```

Run toy experiment:

```
pipenv run python -m examples.run toy
```

## Installation

Install as a package:

```
pipenv install git+https://github.com/pillyshi/pu-learn.git#egg=pulearn
```
